R package for weekly notified dengue cases in Sri Lanka 

## Installation

```R
# Install the development version from GitHub
install.packages("devtools") 
devtools::install_github("thiyangt/mozzie")
library(mozzie)
```

## Example

```R
library(mozzie)
head(mozzie)
summary(mozzie)
```
